# SG-MCMC
Stochastic Gradient Markov Chain Monte Carlo


| Algorithms    | SG-MCMC       | Optimization |
| ------------- |:-------------:| ------------:|
| Basic         | SGLD          |  SGD   |
| Precondition  | pSGLD         |  Adam/RMSprop/Adagrad |
| Momentum      | SGHMC         |  SGD with momentum |
| Thermostat    | SGNHT         |  Santa  |
